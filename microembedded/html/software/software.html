SDR 软件

SDR 软件

软件是研究USRP™软件定义无线电产品线的重要组成部分。我们的目标是为用户提供尽可能多的选择和灵活性，尽可能当谈到开发环境，以及操作系统的支持。我们所有的平台都支持UHD（USRP硬件驱动程序™），确保跨平台代码的可移植性，所以你可以重复使用的大部分新平台上的遗留SDR代码，同时注重新算法的开发。这使您能够最大限度地提高您的工作效率并减少开发工作量。 UHD还支持Linux，Windows和Mac OS，这样你就可以使用研究的SDR平台使用时，您所选择的操作系统。


对于SDR应用软件的开发，所有的USRP特别提款权支持的GNU Radio，软件定义无线电的免费开源软件开发框架。大多数USRP特别提款权也有支持：


  • RFNoC™，从Ettus研究集成到GNU收音机，使FPGA开发，而无需编写VHDL或Verilog的开源软件包。
  • LabVIEW中，一个直观的图形化编程工具，用于管理复杂的系统配置，FPGA的多速率DSP设计和漂浮到定点的转换。
  • MATLAB®和Simulink®中，哪个连接到USRP家族的软件定义无线电是提供一种无线电在半实物环境SISO和MIMO无线系统设计，原型，和验证。

  • RFNoC™(RF Network on Chip)

  • UHD(USRP Hardware Driver™)

  • GNU Radio

  • LabVIEW™

  • MATLAB®/Simulink®
