准备开始

有三种途径可以进入 GNU Radio 世界：

正式稳定发行版：

1.二进制包

2.稳定发行源码包 （tarball）

非稳定版开发用源码包：

3.来自版本管理器（subversion)的非稳定版前端研发用源码包

如果是初次接触 GNU Radio，建议从正式稳定发行版其一着手。

如果这个发行版本具有 ready-to-use 包，最好还是从它着手。它安装容易，也可得到一个到手可用的 GNU Radio。正式发行版提供一个可用于长期开发的稳定代码库。

如果计划中的发行版本或操作系统还没有软件包。也可以从原代码开始生成一个 GNU Radio. 这有点挑战性。从 下载专区 （Download）可以找到下载指导。安装指导包含在 GNU Radio 的发行版本中，也可以从在线的 创建指导（BuildGuide）中得到帮助。

如果想获得最新、最具挑战性的、需要体验一些最近添加的功能，或想对 GNU Radio 做些 社区贡献（contribute），你需要从版本管理器（subversion)库内下载最新的版本。在 创建指导 （BuildGuide）可以得到构建帮助。

若有任何问题和麻烦，发表并在 邮件列表 （discuss-gnuradio mailing list）上提问。在此之前，请先阅读关注 提问及报告错误（ReportingErrors）。

硬件外设

通用软件无线电外设USRP （ Universal Software Radio Peripheral）是推荐的 GNU Radio 的外部接口设备。 USRP 是专门为 GNU Radio 开发的，可以从 Ettus Research 那儿得到。

用 GNU Radio 和其它的 I/O 外设配套是没有问题的。参考 GNU Radio 常见问题（FAQ) 寻求详细的硬件支持资讯。也可以用 USRP 同其它不是 GNU radio 的软件配套。详细请参考 USRP 常见问题（UsrpFAQ）。
